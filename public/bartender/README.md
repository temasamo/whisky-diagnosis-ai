# バーテンダーAI画像ファイル

このディレクトリに以下の画像ファイルを配置してください：

## 必要な画像ファイル

1. **`bartender-background.jpg`** (または `.png`)
   - 用途: チャットボックスの背景画像
   - 推奨サイズ: 1920x1080px 以上
   - 内容: バーカウンター越しに微笑むバーテンダー（男性 or 女性）の画像
   - 説明: チャットボックスの背後に表示され、半透明のオーバーレイでテキストの可読性を保ちます

2. **`bartender-avatar.jpg`** (または `.png`)
   - 用途: アシスタントメッセージの左側に表示するアバター
   - 推奨サイズ: 200x200px 以上（正方形推奨）
   - 内容: バーテンダーの上半身（顔と肩まで）を円形にトリミングした画像
   - 説明: チャットメッセージの左側に48x48pxの円形アイコンとして表示されます

## 画像の配置方法

1. 画像ファイルをこのディレクトリ（`public/bartender/`）に配置
2. ファイル名を上記の通りに設定
3. ブラウザで `/whisky/chat-rag` にアクセスして確認

## 注意事項

- 画像が存在しない場合、アバターは非表示になります（エラーハンドリング済み）
- 背景画像が存在しない場合、デフォルトの背景色（`#3b2b24`）が表示されます
- 画像ファイルの形式は `.jpg`、`.jpeg`、`.png`、`.webp` に対応しています
